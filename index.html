Supermarket E-commerce

Developed an intuitive online shopping platform that allows users to search for and view products, submit reviews, add items to their shopping cart, and securely complete transactions via PayPal or credit card. Users can also track their orders, update profiles, change passwords, and leave feedback. For administrators, the platform offers functionality to easily manage the product catalog, including adding, removing, and editing products. The system emphasizes a smooth, secure shopping experience with a user-friendly design and a focus on accessibility.

Technologies Used:

  • React.js
  • Node.js
  • Express.js
  • MySQL
  • MongoDB

Mercedes Benz Dealership

Designed a responsive website for a Mercedes-Benz car dealership, utilizing HTML, CSS, JavaScript, PHP, and a MySQL database. The site features a dynamic homepage with a car slider that highlights different models and categories. Users can browse various car models, access detailed information, and make informed decisions about their purchases. The website offers a seamless user experience, enhancing customer interaction and streamlining the car selection process.

Technologies Used:

  • HTML
  • CSS
  • JavaScript
  • PHP
  • MySQL

Student Management System

Created a comprehensive student management system using React.js, Node.js, Express.js, and MySQL, aimed at simplifying administrative processes for educational institutions. Students can easily register for semesters, view their profiles, and select lecture groups. The system includes features like displaying lecture locations and schedules, and it allows students to collaborate with peers on group projects. This platform improves academic management and provides an intuitive, user-friendly experience for both students and administrators.

Technologies Used:

  • React.js
  • Node.js
  • Express.js
  • MySQL
